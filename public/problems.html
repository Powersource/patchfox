Patchfox Troubleshooting

For some reason your Patchfox installation is not able to communicate with the running scuttlebot. This can happen for various reasons which are outlined below. Please check each of them.

1. No scuttlebot running

Patchfox uses a local running scuttlebot to connect to the Secure Scuttlebutt network. You can provide it through any of the following options:

  1. By allowing Patchfox to start a local native app. For this to work you need to have followed the setup outlined in Patchfox Host App ReadMe file.
  2. By launching another SSB client that has a bundled scuttlebot such as Patchwork or Patchbay.
  3. By starting your own scuttlebot from the command-line as shown in this page about scuttlebot.

Patchfox can't find the configuration for your sbot

If you're using another SSB client or running your own scuttlebot, then Patchfox can't find the configuration on its own. You will need to go to the Patchfox Settings page in the About Add-ons page and set it up.

Your secret is located in a folder called .ssb in your own users home folder in a file named secret, keep this file safe. Patchfox will try to guess your remote when you paste your secret into the textarea.
